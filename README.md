<h3 id="running">:computer: Project Installation</h3>

<h4>
	<strong>Installing Dependencies</strong>
</h4>

```
yarn
```

or

```
npm install
```

or

```
pnpm install
```

<h4>
	<strong>Running</strong>
</h4>

```
yarn dev
```

or

```
npm run dev
```

or

```
pnpm dev
```
